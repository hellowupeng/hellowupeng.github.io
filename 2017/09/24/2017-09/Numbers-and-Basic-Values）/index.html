Numbers and Basic Values（数值和基本值）

原文地址

用数值，布尔值和其他基础类型模型化数据。

主题（Topics）

逻辑值（Logical Values）

struct Bool

实例是真（true）或假（false）的值类型。

数值（Numeric Values）

struct Int

有符号整数类型。

struct Double

双精度，浮点值类型。

struct Float

单精度，浮点值类型。

范围（Ranges）

struct Range

在一种可比较类型之上的半开区间，从下界到上界，但是不包括下界。

struct ClosedRange

在一种可比较类型之上的闭区间，从下界到上界，包括下界。

错误（Errors）

protocol Error

代表一种能被抛出的错误值。

可选型（Optionals）

enum Optional

代表是一种包裹的值（wrapped）或空（nil）—值的缺失的类型。
