LosslessStringConvertible

Protocol

能用无损的，清晰的方式表示为字符串的类型。

概览（Overview）

例如，整数值1050能用它的全部被表示为“1050”。

一个符合类型的 description 属性必须是原始值的值保护表示。同样的，他应该能从字符串表示重新创建一个实例。
